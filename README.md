Getting started

Clone project:

$ git clone https://github.com/lamjar/LSocial.git
Install dependencies:

$ cd LSocial
$ npm install
Install gem 'sass'

$ gem install sass
Run development web-server:

$ gulp serve

AngularJS

Angular Material
